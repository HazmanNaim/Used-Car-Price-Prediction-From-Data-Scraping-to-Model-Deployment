This Python web app is a demonstration of API implementation using a machine learning model Random Forest Regressor. The model has been trained with a dataset containing 8000 used car listings from the Malaysian market. The web app is built with Flask for the backend and HTML for the frontend.

Please note that this is a test application, and I recommend not relying solely on its predictions for car pricing decisions. For more information and access to the source code, you can visit my GitHub profile.

For reference documentation on Flask, you can visit the Flask documentation, and for HTML, you can refer to the HTML documentation.
